Установка SpinDash
===================

SpinDash использует composer для интеграции в проект. Это означает, что для подключения SpinDash к проекту достаточно создать в корне вашего проекта файл “composer.json” (или отредактировать его, если он уже существует).

.. code-block:: json

	{"require": {"wst/spindash": "dev-master"}, "minimum-stability": "dev"}

Параметр «minimum-stability» в данном случае является обязательным, так как стабильных релизов SpinDash не существует.

При установке будут подгружены некоторые зависимости, в частности, шаблонизатор Twig.

Также обязательным для работы фреймворка является наличие следующих расширений PHP:

* PDO
* Imagick
